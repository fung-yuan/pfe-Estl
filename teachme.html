Project Learning Checklist

Frontend Technologies

  • JavaScript/ES6+
    Core language features, async/await, promises, modules
  • Axios
    HTTP client, interceptors, request/response handling
  • LocalStorage API
    Browser storage for authentication tokens and user data
  • Fetch API
    Native browser API for making HTTP requests
  • Base64 Encoding
    Understanding btoa() and atob() for credential encoding

Backend Technologies

  • Java Basics
    Core Java concepts used in the backend
  • Servlets
    HTTP request handling in Java web applications
  • JPQL
    Java Persistence Query Language for database operations

Authentication System

  • Basic Authentication
    HTTP authentication scheme and Base64 encoding
  • Token Storage
    Managing authentication tokens in localStorage
  • Axios Interceptors
    Automatic token handling in API requests
  • Error Handling
    401 responses and authentication failure management

Natural Language Query System

  • GPT-4 Integration
    How AI processes natural language queries
  • JPQL Generation
    Converting natural language to database queries
  • Special Cases
    Zero absence queries and French character support

Project Architecture

  • Client-Server Architecture
    Understanding the separation of frontend and backend
  • REST API Design
    API endpoints and communication protocols
  • Database Design
    Data models for students, absences, and departments

Security Considerations

  • HTTPS Requirements
    Secure communication for Basic Authentication
  • Token Management
    Secure handling of authentication tokens
  • XSS Prevention
    Cross-site scripting security practices
